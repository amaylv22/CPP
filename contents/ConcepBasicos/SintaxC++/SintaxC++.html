Sintaxis general en C++

  • // → Comienza comentario que se mantiene hasta el final de la línea.

  • ; → Al final de cada instrucción.

  • FUNCION ( ) → Se debe incluir los paréntesis aunque no existan parámetros.

  • /*\*/ → Para el caso de C. Abre y cierra comentarios respectivamente.

  • { → Abre el cuerpo de la función.

  • } → Cierra el cuerpo de la función.

  • " "→ Encierra a las cadenas de texto. (Comillas dobles)

  • ' ' → Encierra letras aisladas (Comillas simples).

  • #INCLUDE <#NAME.h> → Le indica al compilado que es un fichero de (...) del texto y que debe buscarlo entre sus (...).

  • #INCLUDE "NAME.h" → Indica que es un fichero creado por el programa y que lo debe buscar en el mismo (...) en que se guarde el código fuente.

  • INT FUNCTION (INT PAR_1, INT PAR_2) → Si en un fichero se utiliza (...) definido en un segundo fichero, al (...) del primer fichero se debe listar la referencia a la función, sin necesidad de (...) todos sus pasos, solo su nombre y parámetros.

  • : : → Operadores de acceso.

  • & → Operadores de dirección.
