Consejos Básicos del C++

  • Al final de cada renglón debe ponerse un punto y coma (;)
  • Variables: es posible declarar una variable y asignarle valor al mismo tiempo. Por ejemplo, int x=10;
  • Variables: se pueden declarar dos variables del mismo tipo separando una de otra mediante una coma (,), por ejemplo, float y, z;
  • Sintaxis: x=x+1 → x++; o ++x. Si el operador "++" este antes que la variable, el incremento sucederá antes de la operación idicada, si esta después sucederá una vez efectuando la operación indicada. Por ejemplo: 
    x=10;
y=2*(x++);
	Da como resultado:
		x=11
		y=20
    		 
    x=10;
y=2*(++x);
	Da como resultado:
		x=11
		y=22
    		 
    x=10;
x++;
y=2*(x);
  • Operaciones: el operador matemático (/) devuelve resultados diferentes según la variable a la que se asigna, si es el caso de int solo devuelve la parte entera, si es el caso de float devuelve la fracción.
  • Operaciones. Módulos: el operador módulo (%) devuelve el residuo de una división, por ejemplo, REMINDER=x%y;

    Solo funciona con enteros.

  • Comentarios: "//" para una sola línea o "/*, */" para múltiples líneas.
  • Funciones: si la función no regresa ningún valor, entonces es tipo 
    VOID. /// RETURN;
          }
    Entonces la palabra clave RETURN no será igual a ningún valor.
  • Funciones. Main: Por convención la función Main regresa el entero 0 para indicar que el código ha sido ejecutado sin errores, por tanto la función debe ser declarada como entero. 
    INT MAIN()
  • Funciones. Declaración: al declarar las funciones tipo en el encabezado, cada función terminará con ";", mientras que al declarar el cuerpo de la función, esto se omite.
