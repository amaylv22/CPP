Tipos de ficheros

Fichero fuente o código fuente

Archivo de texto que contiene programas en C o C++ con sintaxis comprensible al ser humano, no es un programa ejecutable aún, el texto contenido en este archivo se llama programa fuente.

Fichero objeto, código objeto y compiladores

En el proceso de crear un archivo ejecutable, el programa fuente debe de ser traducido (compilado), para que este sea compresible por el ordenador. El compilado es el programa que se encarga de crear un programa equivalente a nuestro código fuente en el lenguaje del ordenador, este nuevo archivo es llamado fichero objeto y su contenido es el código objeto.

Fichero ejecutable y (...)

Para lograr que nuestro programa corra en el ordenador es necesario utilizar un (...) o "(...)", el objetivo de este programa es agrupar o combinar nuestros ficheros objetos entre sí y con las librerías necesarias, así como con las instrucciones necesarias para (...) los datos (...) memoria y crear el ejecutable.
