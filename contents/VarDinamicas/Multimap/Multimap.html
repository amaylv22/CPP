Multimap

Los multimap son contenedores asociativos como los map, pero que permiten múltiples registros con claves iguales.

Librería de inclusión.

#include <map>

Constructores.

Iguales que para maps, pero se cambia esto por

multimap <TIPO>

Destructores.

[NOM]. ~multimap();

Métodos de Multimaps.

Iguales a map.
Begin Clear Empty End Erase
Get_allocator Insert Key_comp Lower_bound Max_size
Rbegin Rend Size Swap Upper_bound
Value_comp

Propios de multimap.
find (x) Igual que para map pero regresa al iterador del primer elemento con clave = x que encuentra.
equal_range (x) Igual que multiset pero adaptando los iteradores de map, es decir,con first y con second.
count (x) Igual que para multiset pero adaptado a map.

Limpiar contenedores usando el operador =

Para dejar vacío un contenedor ya declarado, usando el operador igual es solo necesario ejecutar la secuencia

[NOM] = <CONTENEDOR> <TIPO(S)> ( );
