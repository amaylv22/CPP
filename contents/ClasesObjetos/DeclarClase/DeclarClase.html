Declaración de una clase

CLASS <IDENTIFICADOR DE CLASE> [<LISTA DE CLASES BASE>]
{	<LISTA DE VARIABLES MIEMBRO>
	<LISTA DE FUNCIONES MIEMBRO>
}	[<LISTA DE OBJETOS A CREAR>];

La <LISTA DE CLASES BASE> es usada solo cuando se direccionan clases, si se omite entonces se declara una clase base.

La <LISTA DE OBJETOS> se incluye cuando se desea crear un objeto de la clase al mismo tiempo de definirlo.

La lista de miembros incluye variables y funciones. Las funciones pueden ser declaradas y definidas dentro de la clase, sin embargo, de ser así las funciones serán tratadas como inline. Lo más recomendable es declarar solo los prototipos de las funciones y definirlas fuera de la clase, para esto se tendrá que referenciar la declaración de la función a la clase a la que pertenece mediante el operador de ámbito (::).
