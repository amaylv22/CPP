Inicializaciones en línea (in line)

 
PAREJA :: PAREJA (INT a2, INT b2)
	{
	a=a2;
	b=b2;
	}
 = 
PAREJA :: PAREJA (INT a2, INT b2):
	a=(a2),
	b=(b2),
	{ }

Esto sirve para definir el constructor al mismo tiempo que se declara dentro de una clase.

Es también posible en la definición de la clase anteponerle a la definición de la función la palabra clave INLINE

INLINE <TIPO> <NOM FUNC> <ARGO>
	{...}
