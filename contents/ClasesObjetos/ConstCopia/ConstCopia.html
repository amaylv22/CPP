Constructor copia

Un constructor de este tipo crea un objeto a partir de otro objeto existente. Estos solo tienen un argumento, que es una referencia a un objeto de misma clase. Crea una copia completa del objeto en el stack. Se utiliza, por ejemplo, cuando el objeto será utilizado como argumento de la función.

<NOM CLASE> :: <NOM CLASE> (CONST NOM_CLASE & OBJ);

Por ejemplo:

CLASS PAREJA
{ PUBLIC:
	// CONSTRUCTOR
	   PAREJA (INT a);
	// CONSTRUCTOR COPIA
	   PAREJA (CONST PAREJA & P);
}

PAREJA : PAREJA (CONST PAREJA & P)
	{ a=P.a;
	  b=P.b
	}
