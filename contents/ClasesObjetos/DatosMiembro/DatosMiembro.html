Clases como datos miembro

Son útiles cuando se requiere combinar distintas clases para crear una más compleja. Por ejemplo:

Se puede entonces crear datos miembro de otras clases, tienen funciones que devuelven datos de tipo de determinada clase.

La forma de acceder a dichos datos es con el operador de acceso (.).

Para ser declarados solamente se antepone dentro de la declaración de una clase el nombre de la clase miembro y el identificador que se desea para el objeto a crear, tal como se ha citado una declaración de una clase.
