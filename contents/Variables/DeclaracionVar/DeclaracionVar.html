Declaración de variables. Más allá de la sintaxis

  • La longitud del identificador al declarar la variable es cualquiera, pero el compilador solamente reconoce los primeros 31 caracteres.

  • Si existen varias variables del mismo tipo se pueden declarar a la vez.

    • INT A;				INT A, B, C;
INT B;		→		ó que
INT C;				INT A,
						B,
						C;

    Es una buena costumbre inicializar las variables locales.

  • Variables con un valor inicial

    • INT A = 234;		INT A=234, B=123, C;

    Array con valor inicial

    INT A[#]=(<%VAL1>, <%VAL2>, ..., <%VAL#>)

    Las variables no inicializadas tienen un valor indeterminado. Es cuando son variables estáticas que se inicializan automáticamente a cero.
