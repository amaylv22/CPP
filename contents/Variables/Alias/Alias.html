Definición de tipos. Alias.

El tipo de una variable nos indica el rango de valores que pueden tomar. En ocasiones puede ser útil definir también para tipos de datos "álias" que hagan más fácil la declaración de variables y faciliten la (...) de los (...).

Sintaxis Ejemplo
VARIABLE 
TYPEDEF [TIPO] <$NOMBRE>;
Declarando una variable de este tipo →		 
TYPEDEF INT ENTERAVAR;
ENTERAVAR CLAVE;
TYPEDEF UNSIGNED CHAR BYTE
ESTRUCTURA 
TYPEDEF STRUCT ([TIPO] <$VAR_1>;
[TIPO]<$VAR_2>;...
<%STRUCT NAME>;
Declarando varios tipos. Asignar valores a la nueva variable.		 
TYPEDEF STRUCT (
	INT VAL;
	FLOAT COST1;
	CHAR ID;) REGISTRO;
REGISTRO EJEMPLO
EJEMPLO.VAL = #;
EJEMPLO. COST = #;
EJEMPLO. ID= #;
PUNTERO 
TYPEDEF <%TIPO> (<$PUNT_NAME>;
<%PUNT_NAME> <&VAL_NAME>;

TYPEDEF puede ser usado también para estructuras, punteros y funciones.
