Cadenas de caracteres

Las cadenas de caracteres no son más que array del tipo char. Estos no pueden ser asignados directamente, con exepción de que este se haga junto con la declaración. La sintaxis se muestra a continuación seguido de algunos ejemplos.

CHAR <VAR_NAME>[<NO_DE_CARACTERES>];
CHAR *<VAR_NAME>;

En el primer caso para asignar memoria estáticamente y en el segundo para asignarla dinámicamente.

Ejemplos.

CHAR TEXTO [5] = "HOLA";
CHAR TEXTO [ ] = "HOLA";

CHAR TEXTO [5];
TEXTO [0] = 'H';
TEXTO [1] = 'O';
TEXTO [2] = 'L';
TEXTO [3] = 'A';
TEXTO [4] = '1000';

Cuando se asigna memoria para cadenas, se debe tomar en cuenta el caracter de fin en la cadena, a pesar de que el vector (...) en la posición 0, se debe de reservar espacio igual a (num, máximo de caracteres) +1.
