Arrays

Permiten agrupar datos usando un mismo identificador. Todos los elementos de un array son del mismo tipo, y para acceder a cada elemento se usan subíndices.

<TIPO> <IDENTIFICADOR> [NUM_DE_ELEMNTOS] [[NUM_DE_ELEM_2]...];

[NUM_DE_ELEMNTOS]
Corresponde al subíndice y tiene valores desde 0 hasta [(NUM_DE_ELEMENTOS)-1].

Declarando arrays con listas

En este caso no es necesario especificar el número de listas (subíndice), ya que se obtienen de los valores de la lista.

<TIPO> <IDENTIFICADOR> [ ] =  {<ELE_1>, <ELE_2>,..., <ELE_>};

Operadores con arrays

Size of

Aplicados a un array, este operador devuelve el tamaño de todo el array en bytes, dado que todos los elementos son del mismo tipo, podemos conocer el número de elementos que forman el array por medio de una sencilla división.

SIZEOF (ARRAY) / SIZEOF (ARRAY [O])
ó
SIZEOF (ARRAY) / SIZEOF (INT)

El segundo caso solo se utiliza cuando el tipo es "INT".
