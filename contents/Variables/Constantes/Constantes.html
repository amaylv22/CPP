Constantes

En C++ es posible aquellas variables que durante el desarrollo del programa y desde su inicio tuvieron el mismo valor como constante.

Existe la recomendación de declarar las constantes con letras mayúsculas y anteponer una "k" minúscula, solo por razones de legibilidad y comprensibilidad.

Las constantes se inicializan al (...).

La forma de declarar las constantes es la siguiente:

CONST [TIPO & MODIFICADORES] <NOMBRE> = <VAL>
#DEFINE <NOMBRE> <VAL>

La diferencia es que la segunda forma define una constante simbólica, es decir, el compilador no usa las variables como tal, si no que sustituye en el código el valor de este antes de armar el ejecutable.

Tipos de constantes

Literales

Corresponden al valor escrito directamente en el código. Ej.15.

Simbólicas

Constante representada por un nombre.

Enumeraciones

Situado entre constante y variable, se define un tipo y a este se le asigna un número finito de valores posibles que puede tomar durante la ejecución. Se describen en el inicio.
