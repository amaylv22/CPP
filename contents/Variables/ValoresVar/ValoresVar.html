Rango de valores de cada variable

 
INT
↓
SHORT
 
UNSIGNED →		0 - 65,535			(2 byes)
SIGNED →		-32768 - 32,767		(2 bytes)
 
BOOL →
 
True or False		(1 byte)
 
LONG INT
 
UNSIGNED →		0 - 4,294,967,295					(4 bytes)
SIGNED →		-2,14,483,648 - 2,147,483,647		(4 bytes)
 
INT
 
UNSIGNED		16 BITS →		0 - 65,535							(2 bytes)
				32 BITS →		0 - 4,294,967,295					(4 bytes)

SIGNED			16 BITS →		-32,768 - 32,767					(2 bytes)
				32 BITS →		-2,147,483,648 - 2,147,483,647		(4 bytes)
 
CHAR →
 
ASCII and ASCII Extended values			(1 byte)
 
FLOAT →
 
1.2e^-38 - 3.4e^38						(4 bytes)
 
FLOAT →
 
2.2e^-308 - 1.8e^308					(8 bytes)

El número de bytes depende del (...) y el computador utilizado.
