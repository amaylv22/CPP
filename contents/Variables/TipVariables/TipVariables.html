Tipos de Variables

Existen solo siete tipos de variables que se complementan con los modificadores para crear gran cantidad de espacios para guardar diferentes tipos de información. Para definir las variables se pueden utilizar solo letras (Mayúsculas y minúsulas), números y el caracter " " con el fin de que el programa sea totalmente transportable. El identificador (a nombre de la variable) se puede (...) con un número y C++ distingue entre mayúsculas y minúsculas.

<#IDENTIFICADOR> = Nombre que le quieres poner.

Tipos de variables

Tipo "char" ó caracter

Variable que contiene un solo dígito (con o sin signo en el caso de los números) o signo de puntuación, tomando 1 byte u octeto. Si no se especifica el modificador de signo, se asume que es con signo. Caracter, incluye números, letras y símbolos.

[SIGNED/UNSIGNED] CHAR <#IDENTIFICADOR>

Se introduce como valores del código ASCII.

Tipo "int" ó entera

Almacena solamente números enteros, de diferentes longitudes como sus modificadores lo indican. Los números pueden ser positivos o negativos.

[SIGNED/UNSIGNED]	[SHORT/LONG/ LONG LONG]		INT
					LONG LONG								<#IDENTIFICADOR>
					LONG						[INT]
					SHORT

Tipo "float" o coma flotante.

Son números con decimales ((...) y exponente) de gran magnitudpero poca precisión. Con un rango de valores de magnitudes no desmedidas es seguro usar hasta 3 decimales.

FLOAT <#IDENTIFICADOR>

Tipo "double" o coma flotante de doble precisión

Son números con decimales de gran magnitud, pero también de gran precisión, su inconveniente es que ocupan mucho más espacio de memoria que las variables tipo float.

[LONG] DOUBLE <#IDENTIFICADOR>

Tipo "bool" ó Booleana

Variables binarias que solo pueden tomar dos valores, true o false.

BOOL <%IDENTIFICADOR>

Tipo "void" o sin tipo.

Indica la ausencia de tipo. Se usa en funciones que no devuelven ningún valor o en las que no requieren parámetros, así como en la declaración de punteros genéricos.

VOID <%IDENTIFICADOR>

Tipo "enum" ó enumerado

Permite definir conjuntos de constantes. Solo podrán tomar los valores definidos. Una vez se asigna un valor o un elemento de la lista, los siguientes toman valores correlativos. Varios identificadores de la lista pueden tener el mismo valor, pero un identificador no puede aparecer en más de un enumerado. Son (...).

ENUM [<%IDENTIFICADOR_DE_ENUM>] {<$NOMBRE>[=<$VALOR>]...<$NOMBRE_>...}

Cada valor-nombre es una nueva constante simbólica, si no se les asigna o indica (...) el valor comienza en 0 y aumenta en ++. Es un (...).

Definido el tipo con constante, se pueden crear nuevas constantes del número, donde se cita el nombre del tipo creado y el nombre de la nueva constante responsable con un espacio.
