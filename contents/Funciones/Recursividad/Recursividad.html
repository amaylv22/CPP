Recursividad

Se dice que una función es recursiva cuando se define en función de sí misma.

Existe la recursividad directa e indirecta.

  • Directa: cuando una función se llama a sí misma.
  • Indirecta: cuando una función A llama a una B que a su vez llama nuevamente a la función A.

Al usar la recursividad se debe incluir un límite para que no se cicle el programa.

Cuando una función se llama a sí misma se crea una nueva capa. Las variables de cada función son diferentes, es decir, se crean variables locales.
