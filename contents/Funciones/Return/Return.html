Return

La sentencia RETURN tiene dos funciones distintas:

  1. Devolver el valor "de retorno" producto de las operaciones propias de la función.
  2. Regresar el control al programa o función desde la cuál fue llamado.

Es por esto que cualquier sentencia escrita después de return "simple" no será ejecutado ni leído por el compilador.

Es posible utilizar más de un RETURN en una función. Por ejemplo, como salida de la función producto de una condición no cumplida.

<TIPO> [NOM_FUNC] ()
	{
	if ()
		RETURN ();
	else
		RETURN ();
	...........
	...........
	...........
	RETURN ();
	}
