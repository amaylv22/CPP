Introducción

Las funciones son un conjunto de funciones que realizan una tarea específica. En general toman valores de entrada llamados parámetros y proporcionan un valor de salida o valor de retorno; aunque tanto unos como otros pueden no existir.

Las funciones deben declararse y definirse. La declaración es solo una presentación mientras que una definición continúa las instrucciones con las que la función realizará su trabajo.

Una función que siempre existe el los programas C++ es la función MAIN( ) (generalmente escrito como INT MAIN ( )), la cual toma el central cuando se ejecute el programa. Dentro de esta (...) el cuerpo de repeticiones del mismo, pero no necesariamente las declaraciones y definiciones de prototipos.

Generalmente la función INT MAIN( ) no tiene valor de retorno especial para cada programa, ete valor indica si el programa ha terminado sin novedad ni errores retornando cero.
