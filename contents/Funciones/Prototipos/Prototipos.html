Prototipos de funciones

En C++ es obligatorio usar prototipos, es decir, una declaración de una función sin cuerpo, solamente con sustituir el valor de retorno y el tipo de sus argumentos, terminando con un ";".

[EXTERN|STATIC] <TIPO> <FUN_NAME> ([<LISTA_DE_PARAMETROS>]);
[<LISTA_DE_PARAMETROS>]
	→ Toma en valor del argumento
	→ Separado por comas para (.....) Por ejemplo, (int largo, int ancho); = C

En un prototipo los nombres de los parámetros con opciones bastan solo con su tipo, pero estos ayudan a la legibilidad del programa.

Cabecera
		<TIPO> <NOMBRE DE FUNCION> ()
		{
Cuerpo
		...
		}

→ Las funciones existentes con accesibles desde cualquier parte del programa, los static solo lo son desde el fichero fuente donde se definen.

→ Si name indica <TIPO> de retorno, el compilador le concederá un (...). Si se quiere que no devuelva nada se usa void.
