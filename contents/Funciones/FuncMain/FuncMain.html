Argumentos de la función Main

Hasta ahora MAIN( ) se había presentado sin argumento alguno, sin embargo existen casos donde es necesario introducirlos, por ejemplo cuando ejecutamos nuestro programa desde la línea de comandos y si deseamos ingresarle algunos parámetros de inicio.

Esto se hace de la siguiente manera:

INT MAIN (<LISTA_DE_PARAMETROS>);

Quedaría quizá más claro con un ejemplo.

INT MAIN (INT ARGC, CHAR **ARGN)

Así, el programa se llama "programa", desde la línea de comando se teclearía:

PROGRAMA ARG1 ARG2 ARG3 ARG4 (Enter)

De esta manera, la variable ARGC valdría 5 (4 argumentos más el nombre del programa que cuenta como un argumento más).

Mientras que ARGU[ ] contendrá la siguiente lisa "<DIRECCION_DEL_EJECUTABLE>", "ARG1", "ARG2", "ARG3", "ARG4".
