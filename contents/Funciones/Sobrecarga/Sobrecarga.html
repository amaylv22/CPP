Sobrecarga de funciones

La sobrecarga en C++ se define por el uso del mismo elemento (funciones, operadores, etc.) para diferentes fines, es decir, que el uso del mismo item produce diferentes objetos según se use.

Así que en C++ se pueden definir varias funciones con el mismo nombre con la única condición que el número y/o el tipo de los argumentos sean distintos. A cada una de las diferentes funciones se les llama versión. La ventaja es evidente cuando es necesario leer las mismas operaciones con objetos de diferente tipo o con diferente número de objetos. Es posible que el tipo o valor de sentencia de la función varie, (int, double, long...), manteniendo el mismo nombre.
