Predicados Binarios

Un predicado binario es una función booleana que viene para hacer comparaciones y devuelve falso o verdadero. Verdadero si la condición se cumple, falso si no se cumple. Recomendando que en los return o if si se ponga.

if (argumento)

Se interpreta como si argumento=true, entonces en los predicados binarios el return se considera una comparación.

return (condición);

Si la condición se cumple regresará true, si no es así, regresará false.

Predicado implementado como función

bool <comparación> (double first, double second)
	{return (int (first) == int(second));}

bool <valor_de_un_solo_digito> (const int & value)
	{return (value <10);}

Predicado implementado como clase

Clase is_odd
	{public:
		bool operator() (const int & value)
	};
