Tipos de variables como parámetros de funciones

Punteros como parámetros

Si pasamos un puntero por valor dentro de la función se modifica el valor apuntado por el puntero y los cambios permanecen al abandonar la función. Sin embargo, los cambios en el propio puntero son locales y no se conservan al regresar. Al pasar por referencia se produce el efecto ya conocido. Al paso se da la dirección de memoria de la variable, con el operador address of. Por ejemplo:

INTERCAMBIAR (&x, &y)

Arrays como parámetros

En realidad, al pasar el nombre del array (1 dimensión) como parámetro, lo que se está haciendo es pasar un puntero al primer elemento, por lo tanto, las modificaciones que se hagan en los elementos del array dentro de la función serán válidos al retornar.

Para el caso de array de varias dimensiones, si solo pasamos el nombre no podremos acceder a los elementos del array mediante subíndices, ya que la función no brinda información sobre el tamaño de cada dimensión. En este caso, para tener acceso se debe declarar el parámetro con un array (con corchetes) "[ ]".

Estructura como parámetros

Las estructuras también pueden ser pasadas por valor y por referencia. El comportamiento es el mismo que en las variables simples ya mencionadas.
