Condiciones

Las sentencias condicionales o de selección, permiten controlar el flujo del programa, seleccionando distintas sentencias en función de diferentes valores.

If...Else

Implementa la ejecución condicional de una sentencia.

If con varias sentencias.

IF (<CONDICION>)
	{	<SENTENCIA_1>;
	<SENTENCIA_2>;
	<SENTENCIA_3>;
	}

Operador relacional.

== igual
!= desigual

If con else,

IF (<CONDICION>)
	<SENTENCIA>;
ELSE IF (<CONDICION>)
	<SENTENCIA>;
ELSE
	<SENTENCIA>;
  • En el segundo caso también pueden existir los bloques de sentencias.
  • La sentencia if se puede anidar.
  • Dentro de la condición también se pueden declarar variables.

Switch

Esta sentencia también implementa una expresión asociada a la aparición de cierto valor, es muy útil cuando se tienes varios posibles valores que cada uno produce un efecto determinado.

SWITCH (<VAR_NAME>)
	{
	CASE <EXPRESION_CTE_1>: [<SENTENCIA>] [<BREAK>];
	...
	[DEFAULT:[<SENTENCIAS>]]
	}
  • <VAR_NAME> → Corresponde a la variable en donde se está almacenando la información a comparar con los casos.
  • <BREAK> → Si esta se omite, si no precede de un break, el programa sigue comparando los CASE y se puede ejecutar más de una sentencia aunque el CASE no coincida.
  • DEFAULT → Se ejecuta en caso de que (...) lo haya hecho.
