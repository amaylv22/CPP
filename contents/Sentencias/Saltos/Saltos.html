Saltos

Las sentencias de saltos han sido explicadas dentro del tipo inmediato anterior, estos son:

  • Continue
  • Break → Solo funciona con for y while.
  • Go to
  • Return
  • Exit

Sentencia Return

Esta sentencia sale de la función donde se encuentra y devuelve el central (focus) a la rutina que la llamó, opcionalmente con un valor de retorno.

RETURN [(EXPRESION)|(VALOR_DE_RETORNO)]

Por ejemplo:

INT PARTDAD (INT X)
{
IF (X%2) RETURN 1;
RETURN 0;
}
