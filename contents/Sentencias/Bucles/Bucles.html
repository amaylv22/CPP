Bucles

Este tipo de sentencias son el núcleo de cualquier lenguaje de programación, permiten realizar tareas repetitivas. También llamados Loop.

While

La sentencia es ejecutable mientras la condición sea verdadera. Comprueba la condición antes de realizar la sentencia.

WHILE (<CONDICION>)
{	<SENTENCIA_1>;
	<SENTENCIA_2>;
	<SENTENCIA_3>; ...}

Por ejemplo:

WHILE (x<100)
	x=x+1;
		Es igual a
WHILE (x++<100);

Con uso de sentencia nula.

Un while (...) ciclo que podría ser while (TRUE). Salir con break.

Do... while

Igual que la sentencia while, la diferencia es que este comprueba la condición después de ejecutar la sentencia, de modo que se ejecutará al menos una vez. Si la evaluación de la condición resulta verdadera, el ciclo continúa.

DO
{	<SENTENCIA_1>;
	<SENTENCIA_2>;
	<SENTENCIA_3>;
}
WHILE (<CONDICION>);

For

La sentencia se repite un número concreto de veces, desde que una variable tiene un valor inicial hasta que (...) otra final.

FOR	([<VALOR_INICIAL>]; [<CONDICION_DE_REPETICION>]; [<INCREMENTO>])
	{	<SENTENCIA_1>;
	<SENTENCIA_2>;
	<SENTENCIA_3>;
	}

Al ser usado FOR se debe declarar.

El bucle se repetirá mientras que la condición de repetición sea verdadera. El campo de "<VALOR_INICIAL>" puede ser una expresión (i=1) o una declaración, siempre y cuando se inicialize la variable (INT i=0).

Por ejemplo:

FOR (contador=0, CONTADOR<12, CONTADOR ++>

Existen tres comandos adicionales que pueden ser usados en conjunto con el FOR.

Continue

Salta alguna de las repeticiones del bucle anterior, es decir, vuelve al inicio del loop antes de que se ejecuten todas las declaraciones.

FOR (i=0; i<=10; i++)
	{
	IF (i==5) CONTINUE;
	PRINT F ("%d", i);
	}

En este caso, cuando i=5 no se ejecutará la sentencia PRINTF( ) y se pasará a la segunda vuelta automáticamente.

Break

Sale de un bucle For automáticamente. Es decir, sale del loop antes de que se cumpla la condición de salida predefinida.

FOR (i=0; i<=10; i++)
	{
	IF (i==3) BREAK;
	PRINT F ("%d", i);
	}

En este caso, cuando i=3 se sale del bucle, se rompe el ciclo.

GOTO <DESTINO>;
